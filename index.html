Recent Projects

Below is a list of my completed projects. They were created using primarily React, Redux, Node.js, PostgreSQL, JavaScript, CSS, and HTML

NADA P.O.S.

NADA P.O.S. is a team created project for restaurants to allow their waiters remote access to the restaurant's Point-of-Sale system. This allows the waiters to go directly to the tables in their assigned section to take food and drink orders, and then send them directly to the kitchen from their own personal tablets and cellphones. Created with React, Redux, PostgreSQL, Node.js, Semantic UI React, JavaScript, CSS, and HTML.

Weather or Not

Weather or Not is a solo React project that can give local forecasts based on the user's ZipCode input. The main screen will display current weather while the Multi-day and Hourly links will display that information respectively. The site was created using React, Redux, RESTful API, Tailwind CSS, and HTML.

Learn more

Heard That!

HeardThat! is a music exploration tool that allow the user to search for their favorite albums out of thousands and add them to their personal playlist to reference later each album also has its own link to buy artist merch or buy a physical copy of that album right from amazon. Created using JavaScript, jQuery, CSS, and HTML.

Learn more

TEAM-ME

Team-Me is an application for managing multiple teams and multiple players. With Team-Me you'll be able to manage which players are on your team, as well as the roles they will be playing. Players will be able to log in and see what roles they are given in advance of any competition. Verified players will also be able to be a part of many different teams at the same time. Created with JavaScript, PostgreSQL, Node.js, CSS, and HTML.
